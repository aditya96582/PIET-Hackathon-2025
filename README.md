# Kisan Sathi Digital Bharat - AI-Powered Agricultural Platform

## 🌾 Project Overview

Kisan Sathi Digital Bharat is a comprehensive AI-powered agricultural platform designed to empower Indian farmers through cutting-edge technology. The platform integrates multiple AI services, weather data, and real-time market information to provide farmers with intelligent farming solutions.

## 🚀 Key Features

### 🤖 AI-Powered Services

#### 1. **AgriMind AI** - Crop Advisory & Seasonal Demand Prediction
- Comprehensive crop advisory based on location and season
- Seasonal demand prediction and market trends
- Optimal planting time recommendations
- Risk assessment and mitigation strategies
- Fertilizer and irrigation recommendations

#### 2. **AgriPredict** - Predictive Export Market Advisory
- International market price analysis and trends
- Best export destinations and demand forecasting
- Quality requirements for export markets
- Logistics and transportation recommendations
- Price predictions for 3-6 months ahead

#### 3. **SeedSense AI** - Fertilizer & Seed Recommendation Engine
- Best seed varieties for specific soil and climate
- Complete fertilizer schedule (NPK, micronutrients)
- Organic vs chemical fertilizer options
- Application timing and methods
- ROI calculations and cost estimates

#### 4. **FarmSage** - AI-Driven Demand Forecasting
- Local and export market demand analysis
- Seasonal price fluctuations and optimal selling times
- Competition analysis and market positioning
- Government policies impact assessment
- Storage and post-harvest recommendations

## 🔧 Installation & Setup

### Prerequisites
- Node.js 18+ and npm
- Modern web browser
- Internet connection for AI features

### Quick Start
```bash
# Clone the repository
git clone <repository-url>
cd kisan-sathi-digital-bharat

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

### API Configuration
The application comes with universal API keys pre-configured:
- **Gemini AI API Key**: Integrated for all AI features
- **OpenWeatherMap API Key**: Integrated for weather services

No additional setup required - the app works out of the box!
